# bird

Use `bird` to interact with X/Twitter.

- Tweet: `bird tweet "..."`
- Reply: `bird reply <tweet-id> "..."`
- Read: `bird read <tweet-id>`
- Thread: `bird thread <tweet-id>`
- Search: `bird search "query"`
- Mentions: `bird mentions`
- Whoami: `bird whoami`

Confirm before posting publicly.
